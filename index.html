Simple Usability JS

Rendi il tuo sito più accessibile, leggibile e conforme con una sola riga di codice.

🎯 Obiettivi

  • ✅ Interfaccia accessibilità pronta all’uso
  • 🌍 Supporto multilingua (IT, EN, FR, DE, ES)
  • 🔠 Personalizzazione font, spaziature, allineamento
  • 🎨 Modalità colore: contrasto, seppia, notte...
  • 🔎 Maschera di lettura, cursore grande, nascondi immagini
  • ⌨️ Navigazione da tastiera & supporto screen reader

🤝 Contribuisci su GitHub

Il nostro progetto è open-source! Visita il repository su GitHub per clonare il codice, segnalare problemi o contribuire con nuove funzionalità.

🚀 Come si integra

Inserisci questi tag nel tuo sito, prima della chiusura di </body>:

<link href="accessibility.css" rel="stylesheet">

<script>
  window.lw_accessibility_var = {
    lang: 'it',
    title: 'Nome del tuo sito',
    mail: 'info@iltuosito.com'
  };
</script>

<script src="accessibility.js"></script>

Puoi anche generare il file JS dinamicamente tramite PHP.
